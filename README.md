# ChefOS

ChefOS ist ein eigenes Linux-Betriebssystem auf Basis von **Ubuntu Server 24.04 LTS (minimized)**.

Das System wird bewusst **ohne vorinstalliertes GUI** aufgebaut und schrittweise zu einem
modernen, performanten **Wayland-Desktop mit Hyprland** entwickelt.

Ziel ist eine **eigene installierbare ISO**, die als saubere Desktop-Basis auf
